## What I work on

- **Runtime engineering** — porting and reverse engineering legacy animation/physics engines
- **Embedded systems** — firmware, USB HID, microcontrollers (RP2040, ESP32, Arduino)
- **Security tooling** — malware analysis, syscall monitoring via eBPF/strace, sandbox isolation with gVisor and namespaces
- **RF & communications** — link budget analysis, antenna simulation, SDR
- **Procedural physics** — behavioral simulation and ragdoll systems, WebAssembly targets
- **Retro computing** — emulation and preservation of abandoned software
